🐦 Gubbi Content

Static weekly content for the Gubbi kids learning app. This site is read by the app — there is no UI here.

  • manifest.json — latest week + list of weeks
  • weeks/2026-06-22/content.json
  • weeks/2026-06-15/content.json

To publish a new week: add weeks/<date>/content.json, update manifest.json, run node validate.mjs, push.
